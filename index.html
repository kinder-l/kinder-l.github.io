Introduction to Basic CSS
  • CSS is one of the languages we can use to style a web page.
  • CSS stands for Cascading Style Sheets.
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  • CSS can save a lot of work. It can control the layout of multiple web pages all at once.
  • External stylesheets are stored in CSS files.

For example, the layout on this webpage utilise the basic elements, such as;

Headers: <h1></h1> or <header></header> Paragraph: <p></p> Lists: <ul> <li>item_here</li> </ul> Footers: <footer></footer>
The Difference Between Class & ID

These will often do the job, but our basic set of tags doesn’t cover every possible type of page element or layout choice.
We need ID’s and Classes to differentiate and style elements.

As an example, the main heading of this webpage is ID'ed as "heading" and styled in CSS.
ID's are unique and can only be assigned to one element. Then we can manipulate it uniquely to other headings on this page.

<h1 id="heading"></h1> h1 {color: #663399; font-family: monospace; font-size: 55px; font-weight: 700; text-decoration: underline wavy #993333;}

Or if we want to manipulate multiple elements, we can apply class instead. This would style all h1 elements.

<h1 class="heading"></h1> .h1 {enter values here;}

Notice how ID is expressed in CSS using a hastag (#) versus class which is expressed using period (.)

Background

These CSS background properties are used to add background effects for elements.

background-color: #color; background-image: url("link_here"); background-repeat: value_here; background-attachment: value_here; background-position: alignment_here;
Text

The following CSS text properties are used to add effects to your text elements.

color: #color; font-family: font_here; font-size: #px; line-height: #px; text-align: alignment_here; text-decoration: value_here;
Hyperlinks

With CSS, links can be styled in many different ways. Links can be styled with any CSS property.

a:link {color: #color; font-family: font_here; font-size: #px; line-height: #px; background: #color; text-align: alignment_here; text-decoration: value_here;}

In addition, links can be styled differently depending on what state they are in. The four links states are:

  • a:link - a normal, unvisited link
  • a:visited - a link the user has visited
  • a:hover - a link when the user mouses over it
  • a:active - a link the moment it is clicked

a:visited {values_here;} a:hover {values_here;} a:active {values_here;}
